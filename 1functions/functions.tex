\section{Sets \&\ Functions}\label{chap:func}

\subsection{Basic Definitions}\label{sec:funcdef}

When did you first encounter a function? Middle-school? Elementary school? Earlier\ldots How would you define \emph{function} to someone with limited mathematical knowledge? Would you use words like \emph{rule, assign, element, domain, vertical line test,} etc.? How helpful are these to your audience?

\begin{examples}{}{funcbasic}
	How would you explain the idea that the following do or do not represent functions?
	\begin{enumerate}
	  \item $y=x^2$
	  \item\label{ex:funcbasic2} Mon: fish,\ \ Tue: pork,\ \ Wed: fajitas,\ \ Thur: carbonara,\ \ Fri: pizza
	  \item $(3,5)$, $(2,6)$, $(4,2)$, $(3,1)$.
	  \item $x^2=y^2$
	\end{enumerate}
\end{examples}

After considering the examples, perhaps you settle on a semi-formal definition:
\begin{quote}
	\texthigh{A function $f$ is rule assigning exactly one output $f(x)$ to each input $x$}
\end{quote}
Is this a useful definition? In what ways is it imprecise? Does the imprecision matter?
\bigbreak

Of course the answers to these questions depend on your audience! What ideas do you want to convey to your students, and can you do so without overburdening and intimidating them? To begin working towards a more complete picture, we need to think about what the \emph{inputs} and \emph{outputs} are allowed to be. This introduces some vocabulary and requires a small amount of set notation.


\begin{defn}{}{}
	A \emph{set} $A$ is a collection of objects, or \emph{elements.}\footnotemark{} The notation $a\in A$ means that $a$ is an element of $A$, sometimes read `$a$ lies in $A$.' In the abstract, sets are often written upper case and their elements in lower case.\par
	\begin{minipage}[t]{0.77\linewidth}\vspace{-5pt}
		A set $B$ is a \emph{subset} of a set $A$, written $B\subseteq A$, if every element of $B$ is also an element of $A$: that is,
		\[
			b\in B\implies b\in A
			\tag{``If $b$ lies in $B$, then $b$ lies in $A$''}
		\]
		The picture illustrates sets $A,B$ and elements $a,b$ for which $B\subseteq A$, $a\in A$, $b\in B$ and $a\notin B$ ($a$ does not lie in $B$).
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.2\linewidth}\vspace{-6pt}
		\hfill\includegraphics[alt={venn diagram of a subset}]{functions-subset}
	\end{minipage}
\end{defn}

\footnotetext{%
	This is enough for our purposes, though a course in set theory should convince you that this definition is not without its own problems. Selection is always at work\ldots%
}

The reason we need this language when discussing functions is that the inputs and outputs of a function are \emph{elements of sets}. 


\pagebreak[3]


\begin{example}{}{}
	Suppose the elements of a set $A$ are the numbers 1, 3, 5, 7 and 9. The simplest way to write this is using \emph{roster notation}: we list the elements (in any order) between braces
	\[
		A=\{1,3,5,7,9\}
	\]
	Subsets are commonly expressed using \emph{set-builder notation}. 
	Here is a subset of $A$:
	\[
	 	B=\{a\in A:2<a<8\} \tag{``The set of $a$ in $A$ such that $a$ lies strictly between 2 and 8''}
	\]
	In roster notation, $B=\{3,5,7\}$. Can you express $B$ in other ways using set-builder notation?
\end{example}


Several familiar sets of numbers can be expressed using informal combinations of these notations.
\begin{description}\itemsep0pt
	\item[Natural numbers] $\N=\{1,2,3,4,\ldots\}$. For instance, $5\in\N$ but $-3\not\in\N$. Most students think this is the meaning of \emph{number} until they've done \emph{lots} of math.
	\item[Integers] $\Z=\{\ldots,-2,-1,0,1,2,3,\ldots\}$. For instance, $-4\in\Z$ but $\frac 45\not\in\Z$.
	\item[Rational numbers] (fractions) \ $\Q=\bigl\{\frac pq:p\in\Z,\ q\in\N\bigr\}$. For instance $-\frac 67\in\Q$. %; in this case $p=-6$ is an integer, and $q=7$ a natural number.
	\item[Real numbers] $\R$. For instance, $\sqrt 2\in\R$. A formal definition is difficult, though we often informally visualize $\R$ as a \emph{ruler}. \emph{Intervals} are particularly important subsets, e.g.,
	\[
		[-4,\pi)=\{x\in\R:-4\le x<\pi\} \tag{a half-open interval}
	\]
	You should also have informally encountered the notion of \emph{irrationality}: for instance, $\sqrt 2$ and $\pi$ are real numbers but not rational numbers.
\end{description}
The subset relationships between these sets are in the order listed: $\N\subseteq\Z\subseteq\Q\subseteq\R$.
\bigbreak

We now turn to a \emph{very} formal definition of function.

\begin{defn}{}{function}
	The \emph{Cartesian product} of sets $A$ and $B$ is the set of \emph{ordered pairs}
	\[
		A\times B=\big\{(a,b):a\in A,b\in B\big\}
	\]
	A \emph{function} from $A$ to $B$ is a non-empty subset $f\subseteq A\times B$ which satisfies the \emph{vertical line test}:
	\[
		\text{For each $a\in A$, there is a \emph{unique} $b\in B$ such that $(a,b)\in f$} \tag{$\ast$}
	\]
	Instead of writing $f\subseteq A\times B$ and $(a,b)\in f$, we use the more familiar notation
	\[
		f:A\to B\quad\text{and}\quad f(a)=b
	\]
	To a function $f:A\to B$ are associated three useful sets:
	\begin{itemize}
	  \item Domain: $\dom f=A$ is the set of \emph{inputs.}
	  \item Codomain: $\codom f=B$ is the set of \emph{possible outputs.}
	  \item Range: $\range f=\big\{b\in B:b=f(a)\text{ for some }a\in A\big\}$ is the set of \emph{realized outputs.}
	\end{itemize}
\end{defn}

This probably isn't the definition you should give to 10\th{} graders! How much of this is helpful in a given context? Is it important that \emph{you've} been exposed to this definition?


\begin{example*}{\ref*{ex:funcbasic}.\ref{ex:funcbasic2} cont.}{}
	We revisit our food-based example in this formal setting. To properly view this as a function $f:A\to B$, we have to carefully label the constituent sets.
	\begin{gather*}
		A=\big\{\text{Mon, Tue, Wed, Thu, Fri}\big\},\qquad\qquad
		B=\big\{\text{carbonara, fajitas, fish, pizza, pork}\big\},\\[5pt]
		f=\big\{(\text{Mon,\ fish}),\ (\text{Tue,\ pork}),\ (\text{Wed,\ fajitas}), \ (\text{Thu,\ carbonara}), \ (\text{Fri,\ pizza})\big\}
	\end{gather*}
	The domain $A$ should be clear, but we had to make a choice for the codomain $B$: in this case we made it to equal to \emph{range.} Can you suggest a different choice for $B$? Note how the function is simply the list of ordered pairs (day, food). Try the other examples yourself.
\end{example*}


\subsubsection*{Representing Functions}

Functions can be represented in various ways. We illustrate a few in an example.

\begin{example}{}{introfunction}
	We consider the familiar \emph{formula/rule} $f(x)=x^2$ in several contexts.\par
	\begin{minipage}[t]{0.75\linewidth}\vspace{0pt}
	  \emph{Table}\lstsp This presentation is most helpful when the domain is very small.
	  \begin{description}
	  	\item[\normalfont\emph{Arrows}] Also useful when the domain is small. Both this and the table describe $f$ when $\dom f=\{-1,0,1,2,3\}$ and $\operatorname{range}f=\{0,1,4,9\}$.
	  	\item[\normalfont\emph{Graph}] This is the set of ordered pairs $\big\{\big(x,x^2\big):x\in\dom f\big\}$. In the context of the formal definition (\ref{defn:function}), \emph{the graph is the function!}\par
	  	For formulæ whose inputs and outputs are real numbers, two conventions are often observed:
		 	\begin{itemize}
		 	  \item The \textcolor{mRed}{domain} is \emph{implied} to be all values for which the formula makes sense (don't do silly stuff like dividing by zero!).
				\item The codomain is the set of real numbers.
			\end{itemize}
			If no other information is provided, we'd assume that the formula $f(x)=x^2$ represents a function $f:\R\to\R$, in which case the \textcolor{Brown}{range} (the set of realized outputs) is the interval
			\[
				\operatorname{range}f=\{x^2\in\R:x\in\R\}=[0,\infty)
			\]
			\smallskip
			For `calculus functions' like this, the vertical line test ($\ast$) really does involve \textcolor{mPink}{vertical lines}; every vertical line intersects the graph in precisely one point.
			\smallbreak
			In the picture, the \textcolor{mGreen}{dots} are the graph when the domain is the set $\{-1,0,1,2,3\}$ (as described in the table/arrow-diagram).
	  \end{description} 
 	\end{minipage}
  \hfill
 	\begin{minipage}[t]{0.23\linewidth}\vspace{-5pt}
 	  \hfill
 	  $\begin{array}{@{}c|ccccc@{}}
 			x&-1&0&1&2&3\\\hline
 			f(x)&1&0&1&4&9
 		\end{array}$

\end{example}

Are there other ways to represent a function? How might your audience influence your choice? We'll have many more chances to analyze and represent functions as we move on. The challenge here is primarily one of vocabulary. A student has perhaps 15 years---elementary school to early college---to become comfortable with all this language. How would you help a student, regardless of where they are on that journey, to take the next step?

\subsection{Linear Polynomials}

Perhaps the simplest functions are those whose graphs are straight lines. These are the \emph{linear polynomials},
\[
	y=f(x)=mx+c,\tag{$\ast$}
\]
where $m,c$ are constants. Such functions make for very simple models:\footnote{%
	Given experimental data or a physical situation relating two quantities $x$ and $y$, a \emph{linear model} is an linear polynomial ($\ast$). In practice, models only \emph{approximate} to real-world data. Later in the course we'll consider what should be meant by, and how to find, a `good' linear model for approximately linear data.
}%
regardless of the initial input, if we increase $x$ by $\Delta x$, then the output changes by a constant multiple $\Delta y=m\Delta x$.\smallbreak

By this stage of your mathematical career, you should be \emph{very} familiar with linear polynomials. From a teaching viewpoint this can be dangerous. The real challenge is how to explain your understanding to those who don't have it. For instance, some of your earliest forays into algebra likely involved finding equations of the straight line through two given points. Ask a college freshman how to solve such a problem and they'll likely offer some version of the \emph{point-slope formula}. But what is this, and why does it work? More importantly, could you teach it?\smallbreak

Abstract concepts in mathematics are very often just a way of writing down all possible examples simultaneously. In such a case, it is usually better to start with an example first before sharing any general formulæ. Here are two questions you could use to introduce abstract concepts around linear polynomials.


\subsubsection*{1. \ How to graph $y=mx+c$? Why is the graph a straight line?} 

\begin{example}{}{easyline}
	Suppose we want to graph $y=f(x)=\frac 12x+1$.\par
	\begin{minipage}[t]{0.7\linewidth}\vspace{-4pt}
		One way to start is to consider the most obvious point on the graph: plug in $x=0$ to obtain $y=1$ and the point $(0,1)$. Next, consider other values of $x$, for instance integers, and plot these:
			\[
				\begin{array}{c|ccccc}
					x&0&1&2&3&4\\\hline
					f(x)&1&1.5&2&2.5&3
				\end{array}
			\]
			Joining these \textcolor{mGreen}{points}, it certainly appears as if the graph is a \textcolor{Blue}{straight line}. At this stage you may have enough to convince a middle-school student.
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.29\linewidth}\vspace{0pt}
		\hfill\includegraphics[alt={a simple straight line plot}]{line-easyplot}
	\end{minipage}
\end{example}

In the abstract, to graph $y=f(x)=mx+c$,
\begin{itemize}
  \item Find the $y$-intercept: $x=0\Longrightarrow y=f(0)=c$ yields the point $(0,c)$.
  \item Plot a few more points $(1,m+c)$, $(2,2m+c)$, $(3,3m+c)$ and draw the line through them.
\end{itemize}

But what if a student is not convinced? What if you are teaching higher-level students? Just because the graph \emph{looks like} a line doesn't guarantee that it is a line. At this point you might have to appeal to some elementary geometry: \emph{similar triangles.}\smallbreak

Plot three points: the $y$-intercept $(0,c)$, the point $(1,1+c)$, and a general point $(x,mx+c)$, and complete two right-triangles as pictured. These triangles are \emph{similar.} Why?\par
\begin{minipage}[t]{0.6\linewidth}\vspace{-6pt}
	\begin{itemize}
	  \item Perhaps your students observe that both are right-angled with the same slope: 
	  \[
	  	\frac{\text{rise}}{\text{run}}=\frac m1=\frac{mx}x
	  \]
	  \item Perhaps they've studied trigonometry and say that the left angle in both triangles is $\arctan m$.
	\end{itemize}
\end{minipage}
\hfill
\begin{minipage}[t]{0.39\linewidth}\vspace{0pt}
	\hfill\includegraphics[alt={The SAS similarity theorem shows we have a straight line}]{line-easyplot2}
\end{minipage}\smallbreak
Both observations are just rephrasings of the SAS triangle similarity theorem.\footnote{%
	If two triangles have two pairs of sides in a common ratio (here $m$) and the angle between them congruent (here \ang{90}), then the triangles are similar. This is \emph{much} older (c.\,2300 years) than graphs and equations (400 years).%
}
The upshot is that both triangles share an angle at $c$, whence the three points are collinear.


\subsubsection*{2. \ How to find the equation of the line through two points?} 

\begin{example}{}{mxplusc}
	We find the equation of the \textcolor{Blue}{linear polynomial} $y=mx+c$ passing through the points $A=(1,3)$ and $B=(4,1)$.
	\par
	\begin{minipage}[t]{0.64\linewidth}\vspace{-5pt}
		Starting by plotting the points and sketching the line.\par
		The natural approach is to substitute both given points into the equation to find two relationships between $m$ and $c$
		\[
			\begin{cases}
				3=m+c\\
				1=4m+c
			\end{cases}
		\]
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.35\linewidth}\vspace{-23pt}
		\hfill\includegraphics[alt={finding the equation of a straight line}]{line-line}
	\end{minipage}
	\medbreak 
	
	Your students might be comfortable solving this system of equations algebraically for $m$ and $c$. But algebra is only convincing if you're already comfortable with the language! Instead consider what you are trying to motivate: the \emph{point-slope formula}. We want the slope $m$, so\ldots\smallbreak
	Ask the students how to get from point $A$ to point $B$. Hopefully you get some version of \emph{along three and down two}. But they've just told you the slope!
	\[
		m =\frac{\text{rise}}{\text{run}}
		=\frac{\text{2 down}}{\text{3 along}}
		=-\frac 23
	\]
	
	\begin{minipage}[t]{0.64\linewidth}\vspace{-5pt}
	To get to any other point $P=(x,y)$ on the line, we're back to similar triangles:
	\begin{gather*}
		-\frac 23=m=\frac{\text{rise}}{\text{run}} =\frac{y-3}{x-1}\\
		\implies y=3-\frac 23(x-1)=\frac 13(11-2x)
	\end{gather*}
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.35\linewidth}\vspace{-28pt}
		\hfill\includegraphics[alt={finding the equation of a straight line}]{line-line3}
	\end{minipage}
\end{example}


\pagebreak[3]


This process works for any two given points with distinct $x$ co-ordinates.

\begin{thm}{Point-slope formula}{pointslope}
	If $x_A\neq x_B$, then the straight line through points $A=(x_A,y_A)$ and $B=(x_B,y_B)$ has slope
	\[
		m=\frac{\Delta y}{\Delta x}=\frac{y_B-y_A}{x_B-x_A}
	\]
	and equation
	\[
		y=y_A+m(x-x_A) =y_A+\frac{y_B-y_A}{x_B-x_A}(x-x_A)
	\]
\end{thm}

The details are in Exercise \ref{exs:lineparam}. Depending on their level, the abstract point-slope formula might be too intimidating for your students. Even if they are capable of applying it, you might feel that its derivation is beyond them. Indeed it might be so for some students, but consider several counterpoints:
\begin{itemize}
  \item Once a student has developed comfort with concrete examples, the point-slope formula and its proof help to summarize and unify what they've learned. An abstract discussion can help build confidence by convincing a student that any such problem can be solved the same way.
  \item The most helpful elementary proofs are those which replicate an examples. Exercise \ref{exs:lineparam} involves no trickery; it simply \emph{reinforces the core technique} by applying it in general.
  \item Encouraging students to think abstractly is an overarching learning outcome of mathematics. You might get push-back, but it's part of the job\ldots
\end{itemize}

Another common issue is that students resist examples and clamor for the point-slope formula: ``So we just need to memorize the formula, right?'' In the moment, the desire for \emph{efficiency} often dominates student thinking: they have many demands on their time and are tempted by a short-cut, the promising of knowing how to do a problem without putting in the work necessary to understand how or why it works. Fighting against this is hard, but consider the following.
\begin{itemize}
  \item Effective learning is rarely efficient. Your goal as a teacher is often to force students to think slowly about something, even when they don't want to. 
  \item Memorizing formulae and algebraic algorithms sacrifices geometric intuition. Mathematics is a language for modeling and describing the real world. The pictures, the \emph{geometry}, are what is real.
  \item Mathematics is interconnected. In this case, the fundamental underlying concept of the point-slope formula is SAS similarity. Linking the discussion to this helps motivate the study of and reinforce foundational geometry. 
  \item You are doing a student no favors by letting them get away with a \emph{see this do that} approach to mathematics. By the end of high-school, there are too many concepts to memorize this way. Your job is to build the scaffolding, not paint the wall for them\ldots
\end{itemize}



\subsubsection*{Modeling with straight lines}

Often the challenge of modeling lies in converting a word problem into algebra---don't underestimate how hard students find this! Here is a simple, if disguised, straight line model.
	
\begin{example}{}{beakers}
	Beaker A contains a 300\,ml solution of a 2\% acid. Beaker B contains 400\,ml of the same acid but of unknown concentration. When the beakers are mixed together the result is an acid with concentration 6\%. What was the concentration in beaker B?\smallbreak
	It should seem natural to denote the unknown concentration (beaker B) by $x$. After mixing:
	\begin{itemize}
	  \item The total volume of the solution is $300+400=700$\,ml.
	  \item The solution contains $300\times\frac 2{100}+400x =6+400x$\,ml of pure acid.
	\end{itemize}
	Putting this together, the concentration of the resulting solution is a linear polynomial function of $x$:
	\[
	 C(x)=\frac{6+400x}{700}
	\]
	The problem is now easily solved: $C(x)=\frac 6{100}\Longrightarrow x=\frac 9{100}=9\%$.
\end{example}

While the algebra in the example is straightforward to those with sufficient experience, consider what natural mistakes grade-school students might make with this problem\ldots



\subsubsection*{Parametrized Lines}\phantomsection\label{pg:paramline}

Here is an alternative visualization of a straight line. Imagine placing a \emph{ruler} so that its zero point is at the origin $O=(0,0)$ and its ``1" lies at a point $P=(p,q)$. If $t$ (a real number) is the number on the ruler, then the points on the line have co-ordinates
\[
	tP=(tp,tq) \tag{$\ast$}
\]

\begin{minipage}[t]{0.65\linewidth}\vspace{-8pt}
	To describe the line through points $A$ and $B$, place a virtual ruler so that 0 corresponds to $A$ and 1 to $B$. Slide the ruler so that $A$ moves to the origin $O$: this amounts to \emph{subtracting} the co-ordinates of $A$ from all points on the line. We obtain a \emph{parallel line} through the origin, with $B$ transformed to $P=B-A$. Putting this together with ($\ast$) results in a parametrized description of the line:
	\begin{align*}
		(x,y) &=A+\mathcolor{mMagenta}{t}P
		=A+\mathcolor{mMagenta}{t}(B-A)\\
		&=(1-\mathcolor{mMagenta}{t})A+\mathcolor{mMagenta}{t}B
	\end{align*}
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.34\linewidth}\vspace{0pt}
		\hfill\includegraphics[alt={parametrizing a line by laying a ruler along it}]{line-line2}
	\end{minipage}
	\bigbreak

The points corresponding to various values of $t$ are \textcolor{mMagenta}{marked}.\medbreak

Contrast this description with the linear polynomial approach. One challenge is that a line may be parametrized using infinitely many different rulers (choose \emph{any} two points on the line!), whereas the linear polynomial description is unique. Does the parametrized approach have any advantages? Which description is easier to understand or to work with?\smallbreak

In the Exercises we make sure that the two descriptions of a line correspond. The discussion is little more than the generalization of an example.

\begin{example}{}{}
	The line through points $A=(3,6)$ and $B=(-1,4)$ may be parametrized
	\[
		(x,y)=(1-t)(3,6)+t(-1,4)=\bigl(3-4t,6-2t\bigr)
	\]
	To convert this to a linear polynomial, first solve for $t$ in terms of $x$,
	\[
		x=3-4t\implies t=\frac 14(3-x)
	\]
	Now substitute this into our expression for $y$:
	\[
		y=6-2t=6-\frac 24(3-x) =\frac 12x-\frac 92
	\]
\end{example}

\subsection{Quadratic Polynomials}

Quadratic polynomials are functions of the form $y=f(x)=ax^2+bx+c$ where $a\neq 0$. The simplest is $y=x^2$, the standard parabola opening upwards: it is straightforward to convince students of the shape of its graph just by plotting a few integer points. For more general quadratics, here are some commonly encountered activities:
\begin{enumerate}
  \item Find the \emph{roots/zeros} of $f$, the solutions $x$ to the equation $f(x)=0$.
  \item Sketch the \emph{graph} of the function $f$.
  \item Use quadratic functions to model a real-world problem.
\end{enumerate}

You likely know two methods for finding zeros: factorizing and the quadratic formula, each of which has its problems. With experience it is easy to spot that 
\[
	x^2+2x-15=(x-3)(x+5)=0\iff x=3\text{ or }x=-5
\]
though the required creativity can make this difficult, particularly when coefficients are large. Students often prefer the quadratic formula since it always works, though at the cost of some intimidating algebra. We'll think about factorization shortly. First, we see how \emph{completing the square} lies behind both the quadratic formula and the standard approach to graphing quadratic functions.

\begin{example}{}{quadraticeasy}
	Describe/graph the parabola $y=-3x^2+12x+4$.\par
	\begin{minipage}[t]{0.69\linewidth}\vspace{-5pt}
		Pay attention to the $x$ terms; $-3x^2+12x=-3(x^2-\textcolor{mGreen}{4}x)$. Observe that
		\[
			-3(x-\textcolor{mGreen}{2})^2=-3(x^2-\textcolor{mGreen}{4}x+4)=-3x^2+12x-12
		\]
		gives most of what we want: note how we \emph{divided the \textcolor{mGreen}{$x$-coefficient} by two.} To finish, just tidy everything up
		\[
			y=(-3x^2+12x-12)+16=-3(x-\textcolor{mGreen}{2})^2+\textcolor{mMagenta}{16}
		\]
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.29\linewidth}\vspace{-18pt}
		\hfill\includegraphics[alt={graph of a quadratic polynomial opening downwards}]{poly-quad2}
	\end{minipage}\medbreak
	The parabola therefore opens downwards ($-3<0$) with its vertex (maximum) at $(x,y)=(\textcolor{mGreen}{2},\textcolor{mMagenta}{16})$. Otherwise said, we translate the parabola $y=-3x^2$ to the right by \textcolor{mGreen}{2} and up by \textcolor{mMagenta}{16}.
\end{example}

This is easy, if intimidating, to repeat in general:
\begin{align*}
	ax^2+bx+c&=a\left(x^2+\frac ba x\right)+c
	=a\left[\left(x+\frac b{2a}\right)^2-\frac{b^2}{4a^2}\right]+c\\
	&=a\left(x+\frac b{2a}\right)^2-\frac{b^2-4ac}{4a} \tag{$\ast$}
\end{align*}

\begin{minipage}[t]{0.67\linewidth}\vspace{0pt}
	Completing the square is what tells us how to graph the function! Starting with the standard parabola $y=x^2$:
	\begin{enumerate}\vspace{0pt}
	  \item Vertically scale by $a$
	  \item Shift horizontally by $-\frac b{2a}$
	  \item Shift vertically by $\frac{4ac-b^2}{4a}$
	\end{enumerate}
	Even more is true, for ($\ast$) may easily be solved for $x$: \emph{completing the square is the quadratic formula.}
\end{minipage}
\hfill
\begin{minipage}[t]{0.3\linewidth}\vspace{0pt}
	\hfill\includegraphics[alt={translating and scaling quadratic polynomials}]{poly-quad}
\end{minipage}


\begin{thm}{}{}
	If $a\neq 0$, then $ax^2+bx+c=0\iff x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$
\end{thm}



\begin{example*}{\ref{ex:quadraticeasy} cont}{}
	Our analysis suggests two equivalent methods for finding the roots.
	\begin{enumerate}
	  \item Quadratic formula: with $a=-3$, $b=12$ and $c=4$, we have
	  \[
	  	x=\frac{-12\pm\sqrt{12^2-4(-3)\cdot 4}}{2(-3)} =\dfrac{-12\pm 4\sqrt{3^2+3}}{-6} =4\pm\frac 2{\sqrt 3}
	  \]
	  Note the difficult surd expression and how much work it was to simplify.
	  \item Use the fact that we've already completed the square:
	  \[
	  	-3(x-2)^2+16=0\iff (x-2)^2=\frac{16}3\iff x=2\pm\frac 4{\sqrt 3}
	  \]
	  It is often simpler to complete the square directly!
	\end{enumerate}
\end{example*}


Quadratics are often used as models. For instance, the motion of a projectile can be modeled using quadratic polynomials, an observation going back to at least to Galileo in the early 1600s: the distance travelled by a falling body is proportional to the \emph{square} of the time taken $y(t)-y(0)\propto t^2$.

\begin{example}[breakable=false]{}{quadseqgraph}
	A body is dropped from a height of 125 meters, taking exactly 5 seconds to reach the ground. Its height at time $t$ seconds is given by $y(t)=125-5t^2$\,m.\par
	\begin{minipage}[t]{0.7\linewidth}\vspace{-8pt}
		This certainly fits with Galileo's observation: $y(t)-y(0)=-5t^2$.\par
		Over each 1s interval, consider how far the body falls:
		\[
			\begin{array}{c|ccccccccccc}
				t&0&&1&&2&&3&&4&&5\\\hline
				y(t)&125&&120&&105&&80&&45&&0\\\hline
				y(t)-y(0)&0&&-5&&-20&&-45&&-80&&-125\\\hline
				\Delta y&&\makebox[0pt][c]{$-5$}&&\makebox[0pt][c]{$-15$}&&\makebox[0pt][c]{$-25$}&&\makebox[0pt][c]{$-35$}&&\makebox[0pt][c]{$-45$}
			\end{array}
		\]
		Since each interval has duration 1\,s, each $\Delta y$ is the \emph{average speed} of the body over that interval.
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.29\linewidth}\vspace{-8pt}
		\hfill\includegraphics[alt={graph of a falling body against time}]{poly-quad5}
	\end{minipage}
\end{example}

You've likely met such problems in calculus and are sorely tempted compute the \emph{velocity} $y'(t)=-10t$\,m/s and \emph{acceleration} $y''(t)=-10$\,m/s$^2$ by differentiation. Historically, and in introductory calculus, it is problems like these that \emph{motivate the definition} of the derivative: note how the last line of the table really suggests that the instantaneous velocity is a linear function!\par
Armed with calculus, Galileo's observation is that the height $y(t)$ solves a differential equation.
\[
	\diff[^2y]{t^2}=-g
\]
where $g$ (approximately 32\,ft/s$^2$ or 10\,m/s$^2$) is the constant acceleration due to gravity. This can, of course, be solved by integrating twice:
\[
	y'(t)=-gt+v_0\implies y(t)=-\frac 12gt^2+v_0t+h_0 \tag{$\ast$}
\]
where the constants of integration are the initial height $h_0$ and vertical velocity $v_0$. Unless you are explicitly teaching calculus or Newtonian physics, this is probably \emph{not} the place to start.

			\emph{Horizontal}\lstsp Since there is no horizontal acceleration, the horizontal speed must be constant: $x(t)=pt+q$ is a \emph{linear function} of time.\smallbreak
			\emph{Vertical}\lstsp Applying ($\ast$) with $g=10$\,m/s$^2$, we see that $y(t)=-5t^2+rt+s$ is a \emph{quadratic function} of time.
		\end{quote}
		Substituting $t=\frac{x-q}p$ into $y$ yields a quadratic function for the \textcolor{Blue}{trajectory}
		\[
			y(x)=ax^2+bx+c
		\]
	\end{minipage}

	\end{minipage}
	
	The details of the solution are left to Exercise \ref{exs:frisbee}. For the present, consider why you should expect \emph{two answers} to the problem. Can you explain why \emph{without} explicitly finding the solutions?

\subsection{Polynomials, Factorization \& the Rational Roots Theorem}

Recall our simple example of factorization in the previous section
\[
	x^2+2x-15=(x-3)(x+5)=0\iff x=3\text{ or }x=-5
\]
That this approach provides \emph{all} roots relies on several familiar algebraic facts:
\begin{enumerate}
  \item Factor Theorem: $f(c)=0\iff x-c$ is a \emph{factor} of $f(x)$.
  \item No zero-divisors: $g(x)h(x)=0\iff g(x)=0$ or $h(x)=0$. 
  \item A quadratic has \emph{at most two} distinct roots.
\end{enumerate}
We'll examine this more closely at the end of this section. For students first learning factorization, it isn't the \emph{why} that's the challenge, it's the \emph{how.} Multiplying out $(x-3)(x+5)$ is mechanical, but factorizing requires some creativity; we can't really factor without somehow knowing that 3 and $-5$ are roots! Beyond making a lucky guess, how might we go about this? 

\begin{example}{}{ratroots1}
	Let's re-examine $f(x)=x^2+2x-15=0$ in a couple of stages.
	\begin{description}
		\item[\normalfont\emph{Integer solutions}] The simplest type of root would be an \emph{integer} $n$. If $f(n)=0$, observe that
		\begin{align*}
			n^2+2n-15=0&\implies n(n+2)=15\implies \text{15 is divisible by $n$}%\\
			%&\implies n=\pm 1,\pm 3,\pm 5,\pm 15
		\end{align*}
		There are only \emph{eight possible candidates} for $n$, and it doesn't take long to test them all:
		\[
			\begin{array}{c|cccccccc}
				n&1&-1&\textcolor{mRed}{3}&-3&5&\textcolor{mRed}{-5}&15&-15\\\hline
				n+2&3&1&\textcolor{mRed}{5}&-1&7&\textcolor{mRed}{-3}&17&-13%\\\hline
				%n(n+2)=15\text{?}&-12&-17&\textcolor{red}{0}&-12&20&\textcolor{red}{0}&240&180
			\end{array}
		\]
		Rather than computing $f(n)$ explicitly, we listed all divisors of $n$ in the first, the corresponding $n+2$ in the second, and mentally checked when $n(n+2)=15$. There are precisely two integer solutions, namely $n=3$ and $n=-5$.
		\item[\normalfont\emph{Rational Solutions}] If you already believe that a quadratic polynomial has \emph{at most} two solutions, then you're done. The next simplest possibility, however, is that a solution be a \emph{rational number} $\smash{x=\frac pq}$: we may assume this is in \emph{simplest terms}.\footnotemark{} Substituting into the polynomial, we see that
		\[
			\frac{p^2}{q^2}+2\frac pq-15=0\iff p^2+2pq-15q^2=0
		\]
		Remembering that $p,q$ are \emph{integers,} we rearrange this equation in two ways:
		\begin{description}
		  \item[$\textcolor{Blue}{p}(p+2q)=15q^2$]\lstsp Since the \textcolor{Blue}{left side} is a multiple of $p$, so also is the \emph{right.} Since $p,q$ have no common factors, it follows that $p$ divides into 15 (15 is a multiple of $p$).
		  \item[$p^2=\textcolor{mRed}{q}(15q-2p)$]\lstsp Since the \textcolor{mRed}{right side} is a multiple of $q$, so also is the \emph{left.} Since $p,q$ have no common factors, we conclude that $q=1$.
		\end{description}
		The upshot is that the only rational solutions to $f(x)=0$ are the two \emph{integers} we've already found.
	\end{description}
\end{example}

\footnotetext{I.e., $p\in\Z$ and $q\in\N$ have no common factors: $\gcd(p,q)=1$.}


\begin{defn}{}{}
	A \emph{degree $n$ polynomial\footnotemark} is any function of the form
	\[
		f(x)=a_nx^n+a_{n-1}x^{n-1}+\cdots+a_1x+a_0
	\]
	where the \emph{coefficients} $a_k$ are constants with $a_n\neq 0$.
\end{defn}

\footnotetext{%
	A quadratic polynomial has degree 2 and a linear polynomial $mx+c$ degree one (if $m\neq 0$). A non-zero constant polynomial has degree zero. By convention, the \emph{zero polynomial} $y\equiv 0$ has degree $-\infty$ so that the theorem $\operatorname{deg} fg=\operatorname{deg} f+\operatorname{deg} g$ makes sense for all polynomials.%
}

Our analysis in Example \ref{ex:ratroots1} generalizes to a famous result.

\begin{thm}{Rational Roots}{}
	Suppose $f(x)=\textcolor{Blue}{a_n}x^n+\cdots+\textcolor{dRed}{a_0}$ has \emph{integer} coefficients where $a_n$ and $a_0$ are non-zero. If $x=\frac pq$ is a rational root in simplest terms, then $\textcolor{Blue}{q}$ divides into $\textcolor{Blue}{a_n}$ and $\textcolor{dRed}{p}$ into $\textcolor{dRed}{a_0}$.\par
	In particular, if $\textcolor{Blue}{a_n=1}$, then the only possible rational roots are integers.
\end{thm}

While you are unlikely to teach the rational roots theorem to grade-school students \emph{as written}, it is the reason why every trick or shortcut used to factorize works! 

\begin{proof}
	Substitute $\frac pq$ into $f(x)$ and multiply by $q^n$ to obtain an equation where everything is an \emph{integer}\vspace{-11pt}
	\[
		\underbrace{\mathcolor{Blue}{a_np^n}+a_{n-1}p^{n-1}q+\cdots+a_1pq^{n-1}}_{\text{divisible by $\mathcolor{mRed}{p}$}} \hspace{-135pt} \overbrace{\phantom{a_{n-1}p^{n-1}q+\cdots+a_1pq^{n-1}}+\mathcolor{mRed}{a_0q^n}}^{\text{divisible by $\mathcolor{Blue}{q}$}}=0
	\]
	By considering the braced terms we see that $\mathcolor{Blue}{a_np^n}$ is divisible by $\mathcolor{Blue}{q}$ and $\mathcolor{mRed}{a_0q^n}$ by $\mathcolor{mRed}{p}$. Since $p,q$ have no common factors, we obtain the result.
\end{proof}


\begin{examples}{}{}
	We factorize two polynomials using the rational roots theorem.
	\begin{enumerate}
		\item If $x=\frac pq$ is a rational root of $f(x)=2x^2-x-3$ in lowest terms, then $q=1$ or 2 and $p=\pm 1$ or $\pm 3$. The eight possibilities for $x$ are easily checked:
		\[
			\def\arraystretch{1.2}
			\begin{array}{c|cccccccc}
				x&1&\mathcolor{mRed}{-1}&3&-3&\frac 12&-\frac 12&\mathcolor{mRed}{\frac 32}&-\frac 32\\\hline
				2x-1&1&\mathcolor{mRed}{-3}&5&-7&0&-2&\mathcolor{mRed}{2}&-4
				%x(2x-1)=3\text{?}&&\checkmark&5&-7&0&-2&\mathcolor{red}{2}&-4\\\hline
				%f(x)&-2&\mathcolor{red}{0}&12&18&-3&-2&\mathcolor{red}{0}&3\\\hline
			\end{array}
		\]
		You may prefer to compute $f(x)$ directly: as in the previous example, since we already know $x$ it is quicker to check whether $x(2x-1)=3$ rather than $f(x)=0$. The two roots are \textcolor{mRed}{indicated}; it is easily verified that the polynomial can be factorized
		\[
			f(x)=(2x-3)(x+1)
		\]
	
	  \item If the cubic polynomial $f(x)=x^3-2x^2+5$ had any rational roots, the only possibilities would be $\pm 1$ or $\pm 5$. It is quickly verified that none of these work,
	  \[
	  	f(1)=4,\quad f(-1)=2,\quad f(5)=80,\quad f(-5)=-170
	  \]
	  whence $f(x)=0$ has no rational roots.
	\end{enumerate}
\end{examples}

Unless there are very few candidates, checking all possibilities is time-consuming. The rational roots theorem is typically used in conjunction with factorization by providing options for how to start. %This still isn't easy, as the next example shows.



\begin{example}{}{factoreasy}
	Given $f(x)=x^3-x^2-7x+10$, the rational roots theorem offers eight candidates: $x=\pm 1,\pm 2,\pm 5,\pm 10$. It is not difficult to check the first few in your head, e.g.
	\[
		f(2)=8-4-14+10=0
	\]
	By the factor theorem, $x-2$ is a factor of $f(x)$. The factorization can be performed in various ways. Here are three options: these, and every other method, are versions of the same process.
	\begin{description}
		\item[\normalfont\emph{Long/synthetic division}]\lstsp You should have practiced this in high-school.
	  \[
	  	\polylongdiv{x^3-x^2-7x+10}{x-2} \implies x^3-x^2-7x+10=(x-2)(x^2+x-5)
	  \]
	  \item[\normalfont\emph{Multiply out and solve}]\lstsp Write $f(x)=(x-2)q(x)$ where $q(x)=ax^2+bx+c$ is some quadratic:
	  \[
	  	x^3-x^2-7x+10=(x-2)(ax^2+bx+c) =ax^3+(b-2a)x^2+(c-2b)x-2c
	  \]
	  Equating coefficients, we obtain the same factorization as before:
	  \[
	  	a=1,\qquad b-2a=-1\implies b=1,\qquad -2c=10\implies c=-5
	  \]
	  
	  \item[\normalfont\emph{Term-by-term factorization}]\lstsp With practice, you can construct the required quadratic factor term-by-term as you go without showing any work. \emph{Teaching} such an approach is likely a terrible idea unless your students are already very comfortable\ldots
	  \begin{enumeratea}
	    \item To create $\mathcolor{mRed}{x^3}$, the first term of the quadratic factor must be $\mathcolor{mRed}{x^2}$ 
	    \[
	    	\mathcolor{mRed}{x^3}\mathcolor{Blue}{-x^2}\mathcolor{mGreen}{-7x}\mathcolor{Purple}{+10}=(x-2)(\mathcolor{mRed}{x^2}+\cdots)=\mathcolor{mRed}{x^3}\mathcolor{Blue}{-2x^2}+\cdots
	    \]
	    \item We have $\mathcolor{Blue}{-2x^2}$ but want $\mathcolor{Blue}{-x^2}$. To correct this, add $\mathcolor{mGreen}{x}$ to the quadratic ($\mathcolor{mGreen}{x^2}\mathcolor{Blue}{-2x^2}=\mathcolor{Blue}{-x^2}$):
	    \[
	    	(x-2)(x^2+\mathcolor{mGreen}{x}+\cdots)=\mathcolor{mRed}{x^3}\mathcolor{Blue}{-x^2}\mathcolor{mGreen}{-2x}+\cdots
	    \]
	    \item We have $\mathcolor{Green}{-2x}$ but want $\mathcolor{Green}{-7x}$. To fix, subtract \textcolor{Purple}{5} from the quadratic ($\mathcolor{Purple}{-5x}\mathcolor{mGreen}{-2x}=\mathcolor{mGreen}{-7x}$):
	    \[
	    	(x-2)(x^2+x-\mathcolor{Purple}{5})=\mathcolor{mRed}{x^3}\mathcolor{Blue}{-x^2}\,\mathcolor{mGreen}{-7x}\mathcolor{Purple}{+10}
	    \]
	    \item Since the last term \textcolor{Purple}{10} is correct, the factorization worked!
		\end{enumeratea}
	\end{description}
	
	You might have seen other approaches involving arranging the coefficients in a table. Regardless, the calculations required to complete these methods are exactly those seen above; all these methods are versions of the same thing.
\end{example}



\subsubsection*{Why Does Factorization Work?}

The theory of factorization relies on some algebra. Here is a \emph{brief} treatment.

\begin{thm}{Factor Theorem}{}
	Suppose $f(x)$ is a degree $n$ polynomial. Then:
	\begin{enumerate}
	  \item $f(c)=0$ if and only if $f(x)=(x-c)q(x)$ for some (degree $n-1$) polynomial $q(x)$.
	  \item The polynomial has \emph{at most} $n$ distinct roots.
	\end{enumerate} 
\end{thm}

\begin{proof}
	\exstart ($\Leftarrow$)\lstsp This is essentially trivial: $f(x)=(x-c)q(x)\Longrightarrow f(c)=(c-c)q(c)=0$.
	\begin{enumerate}\setcounter{enumi}{1}
	  \item[]($\Rightarrow$)\lstsp This relies on the \emph{division algorithm for polynomials}: if $f,g$ are polynomials, then there are unique polynomials $q,r$ with\footnotemark{}
	% \footnote{Let $S=\{f(x)-g(x)q(x):q(x)\in\R[x]\}$. If $0\in S$  stop; $g$ divides $f$. Otherwise, let $k=\min S$ (exists by well-ordering) and let $r(x)=f(x)-g(x)q(x)=a_kx^k+\cdots$ have degree $k$. If $g(x)=b_jx^j+\cdots$ has degree $j\le k$, observe that
	% \[f(x)-g(x)\bigl(q(x)+\tfrac{a_k}{b_j}x^{k-j}\bigr)=r(x)-a_kx^k-\cdots\]
	% has degree $<k=\deg r$: contradiction. Thus $\deg r<\deg g$.\smallbreak
	% For uniqueness, suppose had two candidates $r_1,r_2$, then
	% \[r_1-r_2=g(q_2-q_1)\]
	% But $\deg(r_1-r_2)<\deg g$ and $\deg(g(q_2-q_1))\ge \deg g$ is a contradiction unless $q_1=q_2$ and both sides are zero.}
		\[
			f(x)=g(x)q(x)+r(x)\quad \text{and}\quad \deg r<\deg g
		\]
		If $g(x)=x-c$ is linear, $r(x)$ must be constant. Evaluate both sides at $x=c$ to obtain
		\[
			f(x)=(x-c)q(x)+f(c) \tag{thus $f(c)=0\Longrightarrow f(x)=(x-c)q(x)$}
		\]
		\item Suppose $c_1,\ldots,c_n$ are distinct real roots. By part 1, $f(x)=(x-c_1)q_1(x)$. Since
		\[
			0=f(c_2)=(c_2-c_1)q_1(c_2)\implies q_1(c_2)=0
		\]
		we may factor $x-c_2$ from $q_1(x)$ to obtain
		\[
			f(x)=(x-c_1)(x-c_2)q_2(x),\qquad \deg q_2=n-2
		\]
		Repeat this process to factor out all $n$ linear polynomials $x-c_k$:
		\[
			f(x)=(x-c_1)\cdots(x-c_n)q_n,\qquad \deg q_n=n-n=0
		\]
		whence $q_n\neq 0$ is \emph{constant.} Plainly
		\[
			f(c)=(c-c_1)\cdots(c-c_n)q_n=0\Longrightarrow c=c_j
		\]
		for some $j$, so there are no other roots.\qedhere
	\end{enumerate}
\end{proof}

\footnotetext{%
	For a given example, $q$ and $r$ may be found by synthetic division. This is similar (and may be demonstrated similarly) to the more familiar division algorithm for integers: if $m,n$ are integers, then there are unique integers $q,r$ for which
\[
	m=qn+r\ \text{ and }\ 0\le r<\nm n
\]
In elementary school, this is typically written $m\div n=q\,\textsf{r}\,r$ \ ($q$ remainder $r$); e.g., $23\div 4=5\,\textsf{r}\,3$ corresponds to $23=5\times 4+3$.}


\begin{example*}{\ref{ex:factoreasy} cont}{}
	We know that $f(x)=x^3-x^2-7x+10=(x-2)(x^2+x-5)$. But then
	\[
		f(x)=0\iff x-2=0\ \text{ or }\ x^2+x-5=0
	\]
	The former gives the root $x=2$, and the latter can be attacked via the quadratic formula or completing the square; the polynomial therefore has exactly three real roots
	\[
		x=2,\ \frac{-1\pm\sqrt{21}}2
	\] 
\end{example*}


\begin{example}{}{}
	We finish with a quick example of how long division (or any other factorization method as in Example \ref{ex:factoreasy}) computes the ingredients in the division algorithm.\smallbreak
	If $f(x)=x^3+7x^2-2$ and $g(x)=x^2-2$, then
	\[
		\polylongdiv{x^3+7x^2-2}{x^2-2}
		\implies x^3+7x^2-2=(x^2-2)(x+7)+(2x+12)
	\]
	Otherwise said,
	\[
		f(x)=g(x)q(x)+r(x),
		\text{ where } q(x)=x+7, \ r(x)=2x+12\text{ and }\deg r=1<2=\deg g
	\]
\end{example}

\subsection{Inverse Functions \& the Horizontal Line Test}

The informal idea of an inverse function is that $f^{-1}$ takes the \emph{output} of $f$ and returns its \emph{input} (and vice versa).

\begin{example}{}{easyinvfunc}
	Define a simple function using a table or an arrow diagram\par
	\begin{minipage}[t]{0.74\linewidth}\vspace{-17pt}
		\[
			\def\arraystretch{1.1}
			\begin{array}{c|cccc}
				x&1&2&3&4\\\hline
				f(x)&4&2&5&7
			\end{array}
			\qquad
			\begin{array}{c|cccc}
				y&4&2&5&7\\\hline
				f^{-1}(y)&1&2&3&4
			\end{array}
		\]
		The inverse $f^{-1}$ is the function obtained by \emph{reversing the arrows} or flipping the table upside-down.
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.25\linewidth}\vspace{-13pt}
		\hfill
		\includegraphics[alt={find the inverse by reversing the arrows}]{inverse-easy}
	\end{minipage}
\end{example}


\begin{defn}{}{inversefn}
	A function $f:A\to B$ is \emph{invertible} if it has an \emph{inverse}: a function $f^{-1}:B\to A$ which, for all possible values $x\in A$ and $y\in B$, satisfies
	\[
		f^{-1}\bigl(f(x)\bigr)=x\quad\text{and}\quad f\bigl(f^{-1}(y)\bigr)=y \tag{$\ast$}
	\]
\end{defn}

Certainly Example \ref{ex:easyinvfunc} satisfies the input--output properties ($\ast$). Our concerns are identifying when a function is invertible, how to make it so if not, and how to compute an inverse.

\begin{example}{}{invex1}
	The simple linear function $f:\R\to\R$ defined by $f(x)=2x$ has inverse $f^{-1}(y)=\frac y2$. Its \textcolor{Blue}{graph} admits a similar interpretation to the arrow diagram.\par
	\begin{minipage}[t]{0.75\linewidth}\vspace{0pt}
		\begin{itemize}
			\item The function takes an input $x$, moves it \textcolor{mGreen}{vertically} to the graph, then projects \textcolor{mGreen}{horizontally} to the $y$-axis. This interpretation is precisely the vertical line test (Definition \ref{defn:function})!
			\item The inverse function $f^{-1}$ \emph{reverses the arrows}: project a value on the $y$-axis \textcolor{mMagenta}{horizontally} to the graph, and then \textcolor{mMagenta}{vertically} to the $x$-axis.
		\end{itemize}
		It is unnecessary, but you might want to re-draw the graph of the inverse function so that its domain (inputs) are on the horizontal axis. It is entirely up to you whether to also switch the labels $x$ and $y$.
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.24\linewidth}\vspace{-23pt}
		\hfill

	\end{minipage}
	
	\vspace{-20pt}

Since we are interested in the question of \emph{which} functions can be inverted, it is helpful to consider a function where this arrow-reversing procedure fails.

\begin{example}{}{invexs1}
	Consider $f:\R\to\R$ where $f(x)=x^2-1$. This time, when we attempt to move a real number $y$ horizontally to the graph to compute $x=f^{-1}(y)$, we (usually) encounter one of two problems:\par
	\begin{minipage}[t]{0.71\linewidth}\vspace{0pt}
		\begin{enumeratea}
			\item If $y>-1$, the horizontal line has \textcolor{mMagenta}{two intersections} with the graph, producing  \textcolor{mMagenta}{two choices} of $x$.
			\item If $y<-1$, the horizontal line has \textcolor{BlueViolet}{no intersection} with the graph, so there is \textcolor{BlueViolet}{no way to choose} a suitable $x$.
		\end{enumeratea}
		The naïve approach of \emph{reversing the arrows} fails to define an inverse. However, a simple remedy arises from staring at the graph:
		\begin{itemize}
			\item Problem (a) goes away if we delete the \textcolor{Brown}{left half} of the graph. Otherwise said, we \emph{restrict the domain} of $f$ to $[0,\infty)$. 
			 \item Problem (b) disappears if we insist that $y\ge-1$. Equivalently, we \emph{restrict the codomain} of $f$ to its \emph{range} $[-1,\infty)$. 
		\end{itemize}
	\end{minipage}
	\hfill
	\begin{minipage}[t]{0.28\linewidth}\vspace{-10pt}
		\hfill
		\includegraphics[alt={trying to invert a quadratic function}]{inverses-quad}
	\end{minipage}
	\medbreak	
	After making these restrictions so that $f:[0,\infty)\to [-1,\infty)$, we have a nicely defined inverse function. Indeed
	\[
		f^{-1}:[-1,\infty)\to[0,\infty),\quad f^{-1}(y)=\sqrt{y+1}
	\]
	is easily seen to satisfy the input--output conditions $(\ast)$:
	\begin{gather*}
		x\in [0,\infty)\implies f^{-1}\big(f(x)\big) =\sqrt{(x^2-1)+1}=\sqrt{x^2}=x \tag{since $x\ge 0$!!!}\\
		y\in[-1,\infty)\implies f\big(f^{-1}(y)\big) =\big(\sqrt{y+1}\big)^2-1=y
	\end{gather*}
	We could have made a different choice in part (a) by instead deleting the \textcolor{Blue}{right side} of the graph. This amounts to restricting the domain to $\dom f=(-\infty,0]$ and produces a \emph{different inverse function}. We'll return to this example shortly. The takeaway at present should be the importance of paying attention to \emph{domains} and \emph{ranges}\ldots 
\end{example}



\subsubsection*{What makes a function invertible?}

The fixes in the last example can be rephrased succinctly. A function is invertible precisely when it passes the\ldots  
\begin{quote}
	\texthigh{
		Horizontal line test: every horizontal line intersects the graph exactly once%
	}
\end{quote}
This unpacks to two conditions, each of which addresses one of the problems in the example.


\begin{defn}{}{}
	Let $f:A\to B$ be a function. We say that $f$ is:
	\begin{enumeratea}
		\item \emph{One-to-one} (1--1) if distinct inputs $x_1\neq x_2\in A$ have distinct outputs $f(x_1)\neq f(x_2)$. Otherwise said, there is \emph{at most} one way to achieve a given output: 
	  \[
	  	f(x_1)=f(x_2)\implies x_1=x_2
	  \]
	  If $A,B$ are sets of real numbers with $A$ along the $x$-axis, this is the assertion that each horizontal line intersects the graph \textcolor{dMagenta}{at most once}.
	  \item \emph{Onto} if $\operatorname{range}f=B$. Equivalently,
	  \[
	  	\text{Given $y\in B$, there is some input $x\in A$ for which $y=f(x)$}
	  \]
	  Again, if $A,B\subseteq\R$ with $A$ along the $x$-axis, this says that the horizontal line through $y\in B$ intersects the graph \textcolor{BlueViolet}{at least once}.
	\end{enumeratea}
\end{defn}

Putting these ideas together, a function $y=f(x)$ is both 1--1 and onto precisely when every output $y\in B$ corresponds to a \emph{unique input} $x\in A$. In summary:

\begin{thm}{}{}
	A function $f:A\to B$ is invertible if and only if it is both 1--1 and onto.\smallbreak
	In such a case, it inverse is the function $f^{-1}:B\to A$ such that $f^{-1}(y)=x$ whenever $y=f(x)$.\smallbreak
	Moreover, $\dom f^{-1}=\operatorname{range}f=B$ and $\operatorname{range} f^{-1}=\dom f=A$.
\end{thm}

Let us revisit our previous example in this context.

\begin{example*}{\ref{ex:invexs1}, mk.\,II}{}
	Consider the two properties in the context of the function $f:\R\to\R$ with formula $f(x)=x^2-1$:
	\begin{enumeratea}
		\item Suppose $x_1,x_2$ are real numbers, then 
		\begin{align*}
			f(x_1)=f(x_2)&\implies x_1^2-1=x_2^2-1\implies x_1^2=x_2^2\\
			&\implies x_1=\pm x_2
		\end{align*}
		To force $f$ to be 1--1, it is enough to \emph{restrict the domain} so that all $x$ have the same sign: the natural choice is $\dom f=[0,\infty)$.
		\item Compute the range:
		\[
			\operatorname{range} f=\big\{x^2-1:x\in[0,\infty)\big\}=[-1,\infty)
		\]
		We therefore force $f$ to be onto by \emph{restricting its codomain} to $[-1,\infty)$.
	\end{enumeratea}
	After making these restrictions, the function $f:[0,\infty)\to[-1,\infty)$ is invertible. Its inverse function is obtained by solving $y=x^2-1$ for $x$:
	\[
		x^2=y+1\implies x=f^{-1}(y)=\sqrt{y+1}
	\]
	The \emph{non-negative square root} is used since $x\in\dom f=[0,\infty)$.
\end{example*}



\subsubsection*{An algorithm for inverting functions}

Our discussions suggest an algorithmic process for making a function $f:A\to B$ invertible and for computing an inverse.
\begin{enumerate}
	\item Check that $f$ is 1--1. If not, \emph{restrict the domain} to make it so.
	\item Check that $f$ is onto. If not, \emph{redefine} $B=\operatorname{range} f$.
	\item \emph{Solve} $y=f(x)$ for $x=f^{-1}(y)$.
	\item (Optional) \emph{Switch the symbols} $x,y$ and write $y=f^{-1}(x)$.
\end{enumerate}

The last optional step is because mathematical culture tends to prefer $x$ as an input. If $A,B\subseteq\R$, switching $x\leftrightarrow y$ is equivalent to \emph{reflecting the graph} in the line $y=x$. It is safest to do this step \emph{last}, since you will likely have to consider domains in step 3\ldots\smallbreak

Note that step 1 likely involves a \emph{choice}; depending on how you restrict the domain, you might be able to find multiple inverse functions! To make this explicit, we return once more to our example.



\begin{example*}{\ref{ex:invexs1}, mk.\,III}{}
	Recall that if $f(x)=x^2-1$, then
	\[
		f(x_1)=f(x_2)\implies x_1=\pm x_2
	\]
	Instead of restricting the domain to $[0,\infty)$, we can instead force $f$ to be 1--1 by taking the \textcolor{Brown}{left half} of the graph; that is by \emph{choosing} $\dom f=(-\infty,0]$. The range/codomain remains $[-1,\infty)$, but the inverse function is now different:
	\[
		x^2=y+1\implies x=-\sqrt{y+1}\in (-\infty,0]=\dom f \implies f^{-1}(x)=-\sqrt{x+1}
	\]
	The domain of $f$ forces us to take the \emph{negative square root.}
	\begin{center}
		\begin{minipage}[t]{0.52\linewidth}\vspace{30pt}
			\centering\includegraphics[alt={original graph broken into two domains}]{inverses-combi}
		\end{minipage}
		\begin{minipage}[t]{0.35\linewidth}\vspace{0pt}
			\centering
			\begin{tabular}{@{}c@{}}
				\includegraphics[alt={inverse when the domain of f is positive}]{inverses-combi2}\\[10pt]
				\includegraphics[alt={inverse when the domain of f is negative}]{inverses-combi3}
			\end{tabular}
		\end{minipage}
	\end{center}
	There are other domains on which $f$ is 1--1, but these are the most natural choices. The moral is that you cannot invert a function unless you are precise about its domain and range!
\end{example*}

\begin{example}{}{function-easy}
	We compute inverses for the function with formula $y=f(x)=\frac 1{(x-2)^2}$.\smallbreak
	The implied \textcolor{mRed}{\emph{domain}} is $\R\setminus\{2\}=(-\infty,2)\cup(2,\infty)$: all real numbers except 2.\smallbreak
	The \textcolor{mGreen}{\emph{vertical line test}} is clearly visible on the graph: every vertical line through a point on the domain (except $x=2$) intersects the graph exactly once.\smallbreak
	The \textcolor{BlueViolet}{\emph{range}} is the interval $\R^+=(0,\infty)$: since
	\[
		f(x)=y\iff \frac 1{x-2}=\pm \sqrt y\iff x=2\pm \frac 1{\sqrt y} \tag{$\ast$}
	\]
	any positive output $y$ may be obtained by evaluating $y=f\bigl(2+\frac 1{\sqrt y}\bigr)$.\smallbreak
	The $\pm$ in ($\ast$) is because $f$ fails the \textcolor{mMagenta}{\emph{horizontal line test}}: it isn't 1--1.
	
	\begin{center}
	  \tagpdfsetup{table/header-rows={2}}
		\begin{tabular}{c@{\qquad}c}
			\includegraphics[alt={The graph of the function one over x minus 2 all squared}]{functions-easyex1}
			&
			\includegraphics[alt={The graphs of two possible inverse functions}]{functions-easyex2}
			\\
			The function $f(x)=\frac 1{(x-2)^2}$
			&

		\end{tabular}
	\end{center}

	There are two natural choices for an inverse:
	\begin{enumerate}
		\item Choose \textcolor{blue}{$\dom f=(2,\infty)$}. Then
		\[
		 	\pm\sqrt y=\frac 1{x-2}>0
		\]
		so we take the \emph{positive} square-root. The inverse function is therefore
		\[
			g:(0,\infty)\to(2,\infty),\quad g(x)=2+\frac 1{\sqrt y}
		\] 
		\item Choose \textcolor{Brown}{$\dom f=(-\infty,2)$}. This time
		\[
		 	\pm\sqrt y=\frac 1{x-2}<0
		\]
		We take the \emph{negative} square-root to obtain a second inverse function
		\[
		  h:(0,\infty)\to(-\infty,2),\quad h(x)=2-\frac 1{\sqrt y}
		\] 
	\end{enumerate}
\end{example}

\subsection{Increasing \& Decreasing Functions and Convexity}



Quadratic polynomials are a good excuse to introduce several related concepts.

\begin{defn}{}{}
Let $f:I\to\R$ be a function defined on an interval $I$. We say that $f$ is:
\begin{enumerate}
  \item \emph{Increasing} if $x_0<x_1\implies f(x_0)\le f(x_1)$.
% 	\item \emph{Strictly increasing} if the second inequality is strict; $f(x_1)<f(x_2)$.
	\item \emph{Convex (concave up)} if $x_0\neq x_1\implies (1-t)f(x_0)+tf(x_1)\ge f\bigl((1-t)x_0+tx_1\bigr)$ for all $0<t<1$. Otherwise said, the line segment joining any two points on the graph lies \emph{above} the graph.
\end{enumerate}
Reverse the inequalities to obtain the notions of \emph{decreasing} and \emph{concave down.} The inequalities can also be made strict to obtain \emph{strictly increasing} and \emph{strictly convex,} etc.
\end{defn}

\begin{minipage}[t]{0.45\linewidth}\vspace{0pt}
The function in the picture is convex where its graph is blue. If $0<t<1$, the  value
\[x_t=(1-t)x_1+tx_2\]
lies strictly between $x_1$ and $x_2$. At this point the function lies below the corresponding point on the line joining $(x_1,f(x_1))$ and $(x_2,f(x_2))$:
\[f(x_t)<(1-t)f(x_1)+tf(x_2)\]
\end{minipage}\begin{minipage}[t]{0.55\linewidth}\vspace{0pt}
\flushright\includegraphics{poly-convex}
\end{minipage}\par

\begin{thm}{}{}
If $a>0$, then $f(x)=ax^2+bx+c$ is
\begin{enumerate}
  \item Increasing on the interval $[-\frac b{2a},\infty)$;
  \item Decreasing on the interval $(-\infty,-\frac b{2a}]$;
  \item Concave up on $\R$.
\end{enumerate}
The outcome is reversed if $a<0$.
\end{thm}

Instead of a formal argument, 
we consider the simplest example.

\begin{example}{}{}
$f(x)=x^2$ has apex $(0,0)$. Simply compare\par
\begin{minipage}[t]{0.65\linewidth}\vspace{0pt}
\[f(x_1)-f(x_0)=x_1^2-x_0^2 =(x_1-x_0)(x_1+x_0)\]
and consider two cases:
\begin{itemize}
  \item $0\le x_0<x_1\implies x_0+x_1>0\implies f(x_1)-f(x_0)>0$ so $f$ is \textcolor{blue}{strictly increasing}.
  \item $x_0<x_1\le 0\implies x_0+x_1<0\implies f(x_1)-f(x_0)<0$ so $f$ is \textcolor{Green}{strictly decreasing}.
\end{itemize}
Finally, for any $x_0,x_1\in\R$ and any $0<t<1$, a little algebra shows that
\end{minipage}\begin{minipage}[t]{0.35\linewidth}\vspace{-10pt}
\flushright\includegraphics{poly-quad3}
\end{minipage}

\begin{align*}
f\bigl((1-t)x_0+tx_1\bigr)-(1-t)f(x_0)+tf(x_1)&=\bigl(x_0+t(x_1-x_0)\bigr)^2-(1-t)x_0^2+tx_1^2\\
&=t(t-1)\bigl(x_0+x_1\bigr)^2<0
\end{align*}
whence $f$ is convex (concave up). An alternative argument is in the exercises.
\end{example}
